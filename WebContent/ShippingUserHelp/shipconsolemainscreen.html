Hazardous Material

1. If users want to ship a hazardous item using ShipConsole, they can do this by just checking Hazardous item check box and choosing appropriate Hazardous item type in Package Option Popup.

2. Only after checking the Hazardous Material check box, HazMat Id, Material Type, Class, Quantity, Unit, Identification Number, DOT Label Type, Emergency Contact No, Emergency Contact Name, Packing Group will be enabled otherwise these fields will be disabled.

3. Based on the HazMat Id selected all the remaining fields (i.e., Class, Quantity, Unit, Identification Number, Dot Label Type, Emergency Contact No, Emergency Contact Name, Packing Group) will be populated automatically.

4. Quantity and Unit are applicable only for the FedEx Ground ship method and they are Mandatory fields for this ship method.

5. Other than FedEx ground ship method, quantity and Unit fields are in disable mode.

6. Quantity will also allow decimal values.

7. Possible Unit values are oz, gal, LBS, Lg, qt, kg, pt.

8. Identification Number is applicable only for the FedEx Ground ship method. Type the required DOT Id Number.

9. DOT Label Type is also applicable for the FedEx Ground ship method. Type the DOT Label's required.

10. Emergency Contact Name is applicable only for the FedEx Ground ship method. Enter the Twenty Four Hour Emergency Response Contact Number.

11. Emergency Contact Name is applicable only for the FedEx Ground ship method. Enter the Twenty Four Hour Emergency Response Contact Name.

12. Packing Group is applicable only for the FedEx Ground ship method. Valid Packing Groups are I, II, III.

13. After shipping the Hazardous Material we have to show the extra surcharges for shipping Hazardous Material in the field provided in package option popup. 14. Other than FedEx Ground, all these fields (HazMat Id, Quantity, Unit, Identification Number, DOT Label Type, Emergency Contact No, Emergency Contact Name, Packing Group) will be in disable mode.

FedEx:-


For FedEx shipments when ship button is clicked, request with required values is sent to the FedEx server and we get the Tracking number, Shipment cost and label as response. Multiple package shipment is allowed through FedEx.


Input Validations:-

Below are input validations for which alerts are present and need to follow for the successful shipment.

1) Weight should be Numeric value. For FedEx envelope the weight cannot exceed 10 LBS.

2) UOM should be KG or LB.

3) Dimensions should be Numeric value.

4) COD amount should be Numeric value.

5) For the Pay method Consignee, Freight collect and Third party COD is not allowed.

6) Declared Value:

       a) Should not be greater than $100 for FedEx Ground.

       b) Should not be greater than $500 for Packaging types FEDEXPAK and FEDEXENVELOPE.

       c) If Declared Value selected is more than $500 Signature Option (Direct) Checked automatically.

       d) Declared value should be given in the format N.NN ex: 100.00.

7) Return Shipment package option is not allowed with COD, Future Day shipment and Shipment Method FedExExpressSaver
